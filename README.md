# NonBlock.js

Nonblocking UI elements in JavaScript.

## Installation

Install via NPM with:

```sh
npm install --save nonblockjs
```

Include `NonBlock.css` and `NonBlock.js` in your page.

## Usage

Add the class `nonblock` to any element you want to make nonblocking.
